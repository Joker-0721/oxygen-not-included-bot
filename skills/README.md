# Skills 目錄

本目錄包含《缺氧》自動化項目的 Skills 系統，分為大方向和小目標兩層次。

## 大方向 Skills

| 目錄 | 描述 |
| :--- | :--- |
| [01-environment](./01-environment/) | 環境建構 - 開發環境與調試基礎 |
| [02-perception](./02-perception/) | 感知層 - 世界數據解耦與導出 |
| [03-action](./03-action/) | 行動層 - 指令集與執行 API |
| [04-transport](./04-transport/) | 傳輸層 - 跨語言通訊橋樑 |
| [05-intelligence](./05-intelligence/) | 智控層 - 多代理人決策框架 |
| [06-optimization](./06-optimization/) | 優化層 - 測試與穩健性 |

## 快速使用

AI Agent 使用時，先閱讀大方向目錄的 overview，再根據具體任務選擇對應的小目標 Skills。
